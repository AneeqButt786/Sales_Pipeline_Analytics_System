# Sales Pipeline Analytics System

## Overview

AI-powered sales analytics platform with real-time pipeline visibility, deal tracking, funnel stage monitoring, and recommendations for enterprise and basic plans.

## Features

- Funnel stage monitoring
- Deal value analysis
- Enterprise vs Basic recommendations
- Streamlit UI with sidebar metrics

## Tech Stack

Python 3.12+, OpenAI Agents SDK, Streamlit, python-dotenv

## Setup

1. cd Sales_Pipeline_Analytics_System
2. pip install -r requirements.txt
3. Copy .env.example to .env and set OPENAI_API_KEY

## Run Commands

streamlit run app.py
